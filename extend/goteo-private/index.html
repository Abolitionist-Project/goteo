Goteo Extensions for Goteo.org

This plugin adds custom features to Goteo:

  • Issuing of donation certificates. This includes some extra forms to be filled by donors with their personal data for issuing tax refunds certificates. This only works for Spanish donors, most of the code is “hard-coded” (not configurable) and (may) contain sensitive internal data of the foundation.
  • Public pages with Goteo Foundation particulars (like the Tax calculator page or the organization google’s public calendar view).
  • Views and templates for the matchfunding process. Although internal models contains most of the processing code for multiplying monetary contributions. End user parts (such as the public page of a particular matchfunding call) and the internal administration submodules are not shared publicly. Payment addons and some additional cron processes are also private.
  • An Omnipay’s module for handling payments with the spanish payment gateway used in Goteo.
